Welcome to the Antville Project

Antville is a high performance, full featured weblog hosting software based on [http://helma.org Helma Object Publisher], an open source web application server entirely written in Java. Antville itself is written in !JavaScript and scales very well up to several thousands of weblogs. It is very easy to use while offering a lot of advanced features that make it even capable of hosting other types of websites.

27 June 2014 We migrated the source code repository here at Google Code from Subversion to Git. After trying out [https://www.kernel.org/pub/software/scm/git/docs/git-svn.html Git’s own SVN conversion tool] as well as [https://github.com/nirvdrum/svn2git Git2SVN] and experiencing some issues with revisions 700—866, 3769 and 3770 we did not get any errors with a tool called [http://subgit.com SubGit] – which does not have to mean the issues are really resolved but it provided the easiest approach and the most appealing result.
